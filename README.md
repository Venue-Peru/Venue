# Venue

## Summary

Venue SPA including Landing Page.

## Requirements

- Node.js

## Dependencies

1. Axios: `npm install axios@^1.3.5`
2. PrimeFlex: `npm install primeflex@^3.3.0`
3. PrimeIcons: `npm install primeicons@^6.0.1`
4. PrimeVue: `npm install primevue@^3.26.1`
5. JSON-Server: `npm install -g json-server@0.17.4`
6. Vue-Router: `npm install vue-router@4`
7. Vue-i18n: `npm install vue-i18n@9`

## General Features

1. Router
2. i18n
3. Fake API

## Domain Features

1. Landing Page
2. Login & Signup
3. Profile Management
4. Social Dashboard, Toasts & Clips
5. Product List _of Events_
   1. Search Engine (SEO)
   2. Filter Engine
6. Product Card _of Events_
7. 404 Page Not Found
